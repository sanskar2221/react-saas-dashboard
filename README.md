# React SaaS Dashboard

Modern SaaS analytics dashboard built using:

• React + Vite
• TailwindCSS
• Recharts
• API Driven Architecture
• Responsive Design
• Glassmorphism UI

## Features

✔ KPI Analytics
✔ Dynamic Charts
✔ Orders Table
✔ Global Search
✔ Mobile Responsive
✔ Loading Skeletons
✔ Error Handling

## Tech Stack

React  
TailwindCSS  
Recharts  
Axios  
DummyJSON API
